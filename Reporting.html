Connecting to Google Sheets…

⚠ Google Sheet setup required — the dashboard pulls live data from your Google Sheet, but the sheet must be published first.
In Google Sheets: File → Share → Publish to web → select Entire document → format CSV → Publish.
Then go to ⚙ Settings tab below to confirm the Spreadsheet ID. The sheet stays view-only — no login needed from visitors.

How the join works

1. Parse SAP file
Reads every transaction line: customer code, product code, quantity, UOM and Kshs value.
2. Enrich from Google Sheets (live)
Product code → Brand · Category · Litres/Kg (Products sheet)
Customer code → Channel · Type · Location (Customers sheet)
New rows added to either sheet are picked up automatically on next Refresh.
3. All views rebuild
Channel, Product, Region breakdowns recalculate from the enriched data.

Google Sheets connection

How to publish your Google Sheet (one-time setup)
1. Open the sheet → File → Share → Publish to web
2. Under "Link", choose Entire document and format Comma-separated values (.csv)
3. Click Publish and confirm
4. The Spreadsheet ID is the long string in the URL between /d/ and /edit

Adding new rows to Products or Customers?
Just add rows in Google Sheets and click Refresh in the dashboard — no code changes needed.
